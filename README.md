# 🌐 Hathkhon-Websites

Welcome to **Hathkhon-Websites** 🚀  
This is a multi-page frontend website project built using **HTML and CSS**. It contains different UI pages like login, profile, messages, explore, and more.

---

## 📌 About Project

This project is a practice-based frontend web application.  
It includes multiple connected pages like a mini social/community UI system.

---

## 💻 Technologies Used

- HTML5  
- CSS3  

---

## 📁 Pages Included

This project contains the following pages:

- 🏠 index.html (Home Page)  
- 🔐 login.html (Login Page)  
- 👤 profile.html (User Profile)  
- 💬 message.html (Messages UI)  
- 🔔 notificaton.html (Notifications)  
- 🌍 explore.html (Explore Section)  
- 🧠 aicenter.html (AI Center Page)  
- 📝 createrequest.html (Create Request Page)  
- 📄 detail.html (Details Page)  
- 🏆 leaderboard.html (Leaderboard Page)  
- 🎨 style.css (Main Styling File)

---

## 🎯 Features

- Multi-page website structure  
- Modern UI layout  
- Clean and organized code  
- Beginner-friendly frontend project  
- Simple navigation between pages  

---

## 🚀 How to Run

1. Download or clone this repository  
2. Open the folder  
3. Open `index.html` in your browser  
4. Navigate through pages using links
